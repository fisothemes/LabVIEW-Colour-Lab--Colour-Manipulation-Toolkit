# LabVIEW Colour Lab
A labview toolkit for fast and simple RGB, HSV, HSL, HEX or DEC colour conversion. 

**The Project includes the following tools/VIs:**

- [x] Benchmark Tests
- [x] Colour to RGB
- [ ] HSL to RGB*
- [x] HSV to RGB
- [x] RGB to Colour
- [ ] RGB to HSL*
- [x] RGB to HSV

<sup>* WIP - Work-In-Progress</sup>

# Minimum Requirements
* LabVIEW 2018 (32/64 bit)
